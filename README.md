Project Overview
This project focuses on cleaning and preparing a loan dataset containing customer, income, credit, and loan repayment information.  
The goal was to ensure the dataset is accurate, consistent, and ready for analysis and visualization.

 Dataset Description
- Approximately 20,000 loan records
- 22 columns including customer demographics, income, loan details, and repayment status

 Data Cleaning Steps
- Checked for duplicate records
- Reviewed dataset for missing or blank values
- Validated numerical ranges such as age, income, loan amount, and credit score
- Standardized categorical fields for consistency
- Interpreted loan repayment indicators for clarity
- Created helper columns to support analysis (loan size categories, credit score bands)

 Tools Used
- Microsoft Excel

 Outcome
The cleaned dataset is structured and ready for KPI calculation and dashboard development to support financial and operational insights.
